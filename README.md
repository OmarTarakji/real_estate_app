# Real Estate App (Flutter + Django Backend)

This is a beautiful, production-ready Flutter app for a generic real estate platform, powered by your Django backend. It covers authentication, property browsing, booking, favorites, and user profile management.

## Features
- User authentication (signup, login, logout)
- Browse real estate listings
- Property detail view
- Booking flow
- Favorites system
- User profile & settings
- Modern, clean Material 3 UI
- Ready for Android
